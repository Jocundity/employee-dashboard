# Employee Dashboard

This is a full-stack web application (Vue.js, Express.js, PostgreSQL) designed to be used as an internal productivity tool for employees. It features clock, task management, weather (Open-Meteo API), currecy exchange (Frankfurter API), news (GNews), and AI chat assistant (Google Gemini API) widgets. It also features an AI-powered automation function to break down and schedule large tasks into subtasks. User authentication was accomplished with JSON Web Tokens (JWT).
